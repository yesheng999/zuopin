📁 如何使用

每一类作品下方都有"导出"按钮，点击即可下载对应格式的文件。
导出的文件会保存到浏览器默认下载文件夹中。

推荐：用手机录屏功能打开动画页面全屏播放录制，直接得到MP4。

互联网类 · H5页面二维码

格式：PNG二维码 · 含H5链接 · 评审扫码即可访问

在下方输入 GitHub Pages 部署后的 H5 页面链接，点击生成二维码。
评审人员用手机扫码即可打开互动答题页面。

示例：https://yesheng999.github.io/zuopin/internet/h5.html

📖 GitHub Pages 部署步骤（3步搞定）：
第1步：登录 GitHub（github.com），点击右上角"+"→"New repository"， 仓库名填 zuopin，选 Public，点"Create repository"
第2步：在本页面下方点击"一键推送代码到GitHub"按钮（需先在本机配置GitHub登录）
第3步：在GitHub仓库页面点 Settings → Pages → Source 选"master"分支 → Save， 等待1-2分钟后页面顶部出现链接地址

部署成功后：将生成的链接（如 https://用户名.github.io/zuopin/internet/h5.html） 填入上方输入框，点击生成二维码即可用于投稿。
提示：链接在征集活动期间需保持可访问，请勿删除GitHub仓库。
